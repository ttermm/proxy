# Proxy

### Требования
| Пакет | Версия | Особенности установки|
| ------ | ------ | ------ |
|nodejs|8.7.0|
|npm|4.6.1|
|forever||npm install forever --global|
|git||

### Установка пакетов для Ubuntu
Nodejs
```sh
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Быстрый старт
```sh
npm i
make start
```

### Запуск
```sh
make start
```

### Выключение
```sh
make stop
```
